Rust|环境安装

发表于2022-11-07|更新于2022-12-10

学习操作系统之后对系统编程的好感度逐渐上升，但是对C/C++不是很感冒

听说了 Rust 的大名后，遂来学习，也打算使用这么语言完成学校的操作系统课设

下面先完成 Rust 的环境安装配置吧

Rust官网：安装 Rust - Rust 程序设计语言 (rust-lang.org)

🎐Windows系统安装Rust环境

首先需要下载 rustup-init.exe，完成后，直接运行，但此时会提示你需要安装相关的C/C++环境

如果电脑没有C/C++相关的环境配置的话，有以下两种方式进行安装：

  • 安装官网推荐的 Microsoft C++ 生成工具，然后下载一系列的VC环境
  • 只安装 MinGW，自己进行环境变量的配置

因为 Microsoft C++ 生成工具下载的东西实在太多，难以接受，所以我使用的是第二种方式

第一种方式虽然需要下载很多的东西，但是很适合不想配置变量环境的同学，下载后可以直接运行 rustup-init.exe 进行安装

所以就只记录第二种安装方式的过程

下载MinGW

MinGW是一系列 C/C++ 的编译工具，下载 MinGW for Windows

配置好相应的环境变量，我的电脑->属性->高级系统设置->环境变量

在系统变量的Path中添加 D:\Environment\mingw64\bin

修改 Rust 的安装路径

Rust安装是默认安装在C盘的，如果对此不介意，可以直接跳过

首先在相应的目录下新建.cargo和.rustup文件夹

我这里的是 D:\Amber\Plugins\Rust\.cargo\ 和 D:\Amber\Plugins\Rust\.rustup\

同样需要在环境变量(但不是 Path 中)，添加对应的路径：

名称：CARGO_HOME，值：D:\Amber\Plugins\Rust\.cargo\

名称：RUSTUP_HOME，值：D:\Amber\Plugins\Rust\.rustup\

安装 Rust

终于到正题了，现在可以点击 rustup-init.exe 进行安装了

安装后，会给出三个提示选项，这里因为要自定义安装，所以输入2回车

随后就是安装cargo、rustc、rustup等工具，是否要默认

所以这里要修改一下默认的选项，选择2

选择工具链，这里输入x86_64-pc-windows-gnu回车

选择稳定版本的工具链，输入stable即可

输入y表示修改PATH环境变量：

此时已经修改了默认安装路径了，会找到环境变量中配置的CARGO_HOME和RUSTUP_HOME修改路径，可以输入1进行安装了

这里会下载很多编译工具，等个两三分钟就好
